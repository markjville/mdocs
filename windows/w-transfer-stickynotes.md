# Transferring Sticknotes to new computer:

1. Copy the Stickynotes DB from old computer:
`C:\Users\%username%\AppData\Local\Packages\Microsoft.MicrosoftStickyNotes_8wekyb3d8bbwe\LocalState\plum.sqlite`

2. Install Sticknotes, if not already. Copy or Replace the "plum.sqlite" DB to the same location:
`C:\Users\%username%\AppData\Local\Packages\Microsoft.MicrosoftStickyNotes_8wekyb3d8bbwe\LocalState\plum.sqlite`
