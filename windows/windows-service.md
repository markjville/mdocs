# Windows servicing

1. Ask for product keys and installation medium
- If none, use Belarc Advisor to recover keys.

2. Create System Repair Disk: Control Panel/Backup and Restore/Create a system repair disk

3. Create System Image: Control Panel/Backup and Restore/Create a system image
-Note: this will likely be tens of gigabytes in size.

4. Backup data:
-User data
-Browser bookmarks
-Desktop theme

5. Perform service

6. Verify service

7. Restore user data if applicable
