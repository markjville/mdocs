# Proceedure for a compromised Windows computer

1. Determine make model
2. Determine if there is backup of data and software keys
3. Find out which Fn key to reach BIOS
4. Boot to BIOS, disable WiFi
5. Boot computer to Desktop
6. Determine what are symptoms. Is there a virus? That affects computer or is browser hijacked?
7. Run AV scan

If nothing shows and there is suspicion of infection, 

8. Restore system to previous restore point

9. To reset your PC, go to Start  > Settings  > System  > Recovery . Next to Reset this PC , select Reset PC. Then, select Keep my files, choose cloud or local, change your settings, and set Restore preinstalled apps? to No. 

If problem persists, reinstall system from recovery partition or media
