# Install vobcopy on CentOS 8:

## 1. Download the Vobcopy RPM [here](https://fedora.pkgs.org/29/rpmfusion-free-x86_64/vobcopy-1.2.0-10.fc29.x86_64.rpm.html) Then run:

`sudo rpm -i vobcopy-1.2.0-10.fc29.x86_64.rpm` (or which ever version you downloaded)

Or 

## 2. Download and install latest vobcopy package if you have EPEL repo installed:

`dnf install vobcopy`
